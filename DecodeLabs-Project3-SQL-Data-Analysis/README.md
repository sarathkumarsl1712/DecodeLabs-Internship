# SQL Data Analysis Project-3

## Objective of the Project
Analyze e-commerce sales data using SQL.

## Tools Used for this Project
- MySQL Workbench
- SQL

## Dataset
1200 sales records containing customer, product,
payment and revenue information.

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()

## Key Insights
- Total Revenue: ₹1,264,761.96
- Average Order Value: ₹1053.97
- Product-wise sales analysis
- Referral source performance
- Coupon utilization trends

## Conclusion
SQL helped transform raw sales data into actionable
business insights through filtering, aggregation,
and grouping techniques.
