# Exploratory Data Analysis (EDA) - DecodeLabs Project 2

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) using an e-commerce transactions dataset.

## Objectives
- Clean and analyze the dataset
- Calculate descriptive statistics
- Identify trends and patterns
- Detect potential outliers
- Generate business insights

## Tools Used
- Microsoft Excel
- COUNTIF Functions
- Statistical Analysis

## Key Findings

- Total Records: 1200
- Total Revenue: ₹12,64,761.96
- Average Order Value: ₹1,053.97
- Most Ordered Product: Printer
- Most Preferred Payment Method: Online
- Highest Performing Referral Source: Instagram
- Most Used Coupon: FREESHIP

## Business Insights

- Instagram generates the highest customer acquisition.
- Online payment is the most preferred payment method.
- Cancelled and returned orders are relatively high and require investigation.
- FREESHIP coupon drives significant customer engagement.

## Dataset Information

The dataset contains 1,200 e-commerce transaction records with information such as:

- Product Category
- Order Quantity
- Unit Price
- Payment Method
- Order Status
- Coupon Usage
- Referral Source
- Total Price
